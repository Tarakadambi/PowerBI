# 📊 Bank Customer Churn Analysis (Power BI Project)

## 📖 Overview
This project analyzes **bank customer churn data** using Power BI.  
The goal is to understand why customers leave the bank and identify key factors influencing churn through interactive dashboards and data visualization.

---

## 🚀 What I Did
- Imported and cleaned bank customer churn dataset  
- Created interactive dashboards using Power BI  
- Analyzed customer behavior and churn patterns  
- Identified key factors affecting customer retention  
- Built visual insights for business decision-making  
- Generated a final report in PDF format
